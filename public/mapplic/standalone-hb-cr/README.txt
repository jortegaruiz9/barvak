Standalone package for the HB-CR lots map.

Contents:
- index.html
- mapplic.js
- mapplic.css
- data-hb-cr.json
- assets/maps/hb-cr-interactive.svg
- assets/thumbs/lots/*.svg

Run options:
1) Open with a local static server (recommended):
   - Python: python3 -m http.server 8080
   - Then open: http://127.0.0.1:8080

2) You can also upload this folder to a web host and open index.html.
